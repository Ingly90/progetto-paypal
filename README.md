# Progetto PayPal - Selezione Numeri

## Requisiti
- Node.js installato
- PayPal.me configurato

## Installazione
1. Clona il repository.
2. Installa le dipendenze: `npm install`.
3. Avvia il server: `npm start`.
4. Apri il browser su `http://localhost:3000`.
